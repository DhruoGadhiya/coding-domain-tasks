# ROS 2 (Humble)

ROS 2 is Robot operating system and ROS 2 Humble is a version of ROS 2 used in robotics.

ROS 2 is a software framework used in robotics and It helps in building and controlling robots by connecting different
parts of a robot like sensors, motors, and controllers.

ROS 2 is used because robots have many parts and programs and it helps these programs talk to each other easily.

## Nodes 
Nodes are small programs that has access to ROS functionalities and ROS communication.
There are various types of nodes such as nodes that communicate between each other and print logs on terminal and there are nodes which create graphical interface.
